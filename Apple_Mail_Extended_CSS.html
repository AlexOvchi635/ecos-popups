🍎 Apple Mail - Расширенные CSS возможности

Apple Mail (iOS и macOS) имеет ЛУЧШУЮ поддержку CSS среди всех email-клиентов! Здесь можно использовать современные CSS3 фичи, которые не работают в Gmail, Outlook и других клиентах.

💡 Как использовать эту таблицу:
1. Выделите таблицу ниже (Ctrl+A или мышкой)
2. Скопируйте (Ctrl+C)
3. Вставьте в Google Sheets
4. Используйте эти CSS свойства для создания красивых писем специально для Apple пользователей!

⚠️ ВАЖНО: Все эти продвинутые свойства работают ТОЛЬКО в Apple Mail! Для других клиентов нужны fallback варианты.

💡 Как использовать эти возможности

1️⃣ Создайте "Apple-Only" версию письма

Определяйте Apple Mail и показывайте расширенную версию с градиентами и анимациями:

<style>
  /* Базовые стили для всех */
  .button { background: #3498db; }
  
  /* Улучшенные стили только для Apple Mail */
  @supports (-webkit-background-clip: text) {
    .button-premium {
      background: linear-gradient(135deg, #667eea, #764ba2);
      transition: transform 0.3s;
    }
    .button-premium:active { transform: scale(0.95); }
  }
</style>

2️⃣ Dark Mode поддержка

Обязательно добавьте стили для тёмной темы:

@media (prefers-color-scheme: dark) {
  body { background: #1a1a1a; color: #fff; }
  .card { background: #2a2a2a; }
  img { opacity: 0.9; } /* Изображения немного тусклее */
}

3️⃣ Градиентный текст (WOW-эффект!)

.gradient-text {
  background: linear-gradient(135deg, #667eea, #764ba2);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  font-size: 48px;
  font-weight: bold;
}

4️⃣ Анимированные кнопки

.animated-button {
  background: linear-gradient(135deg, #667eea, #764ba2);
  transition: all 0.3s ease;
  box-shadow: 0 4px 15px rgba(102, 126, 234, 0.4);
}
.animated-button:active {
  transform: translateY(2px) scale(0.98);
  box-shadow: 0 2px 8px rgba(102, 126, 234, 0.4);
}

5️⃣ Fallback для других клиентов

⚠️ ВАЖНО: Всегда делайте запасные варианты!

.button {
  /* Fallback для Gmail/Outlook */
  background-color: #3498db;
  
  /* Градиент для Apple Mail */
  background: linear-gradient(135deg, #667eea, #764ba2);
}
