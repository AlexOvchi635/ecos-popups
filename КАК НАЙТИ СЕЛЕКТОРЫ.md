# 🔍 КАК НАЙТИ СЕЛЕКТОРЫ НА СТРАНИЦЕ ECOS

## 📋 ИНСТРУКЦИЯ ПО ПОИСКУ СЕЛЕКТОРОВ

---

## ШАГ 1: Открой страницу ECOS Marketplace

1. Открой: `https://ecos.am/en/asics-marketplace/`
2. Нажми `F12` (откроется DevTools)
3. Нажми на значок **🔍 Inspector** (или `Ctrl+Shift+C`)

---

## ШАГ 2: Найди элементы для тура

### **🎯 ШАГ ТУРА 1: Приветствие (центр экрана)**
**Не требует селектора** - показывается по центру

---

### **🎯 ШАГ ТУРА 2: Первая карточка товара**

**Как найти:**
1. Наведи инспектор (🔍) на **первую карточку товара**
2. Кликни на неё
3. В DevTools увидишь что-то типа:

```html
<div class="product-card">
  <h3>Antminer S19 Pro</h3>
  <div class="price">$2,499</div>
  <button>Buy</button>
</div>
```

**Какой селектор использовать:**
- Если есть класс `product-card` → `.product-card:first-of-type`
- Если есть класс `miner-card` → `.miner-card:first-of-type`
- Если есть ID `#miner-1` → `#miner-1`

**В коде уже есть резервные варианты:**
```javascript
selectors: [
    '.product-card:first-of-type',
    '.product:first-of-type',
    '[class*="product"]:first-of-type',
    '[class*="miner"]:first-of-type',
    '[class*="asic"]:first-of-type',
    '.card:first-of-type',
    'article:first-of-type'
]
```

---

### **🎯 ШАГ ТУРА 3: Кнопка "Купить"**

**Как найти:**
1. Наведи инспектор на кнопку **"Buy"** / **"Купить"**
2. Кликни
3. Увидишь:

```html
<button class="buy-btn">Buy Now</button>
<!-- ИЛИ -->
<button class="add-to-cart">Add to Cart</button>
<!-- ИЛИ -->
<a href="/cart" class="btn-primary">Buy</a>
```

**Какой селектор использовать:**
- Если кнопка: `button.buy-btn`
- Если ссылка: `a.btn-primary`
- Если есть data-атрибут: `[data-action="buy"]`

**В коде уже есть резервные варианты:**
```javascript
selectors: [
    'button[class*="buy"]',
    'button[class*="add"]',
    '.buy-button',
    '.add-to-cart',
    'a[class*="buy"]'
]
```

## ШАГ 3: Проверка селектора

### **В консоли DevTools (F12 → Console):**

```javascript
// Проверь, находит ли селектор элемент:
document.querySelector('.product-card:first-of-type')

// Должен вернуть элемент (не null):
// → <div class="product-card">...</div>

// Если вернул null - селектор неправильный
```

## 🎯 ПРИМЕРЫ РЕАЛЬНЫХ СЕЛЕКТОРОВ

### **Частые варианты:**

| Элемент | Возможный селектор |
|---------|-------------------|
| Карточка товара | `.product-card`, `.miner-item`, `.asic-card` |
| Кнопка покупки | `.btn-buy`, `.add-to-cart`, `button.primary` |
| Поиск | `input[type="search"]`, `.search-input` |
| Фильтры | `.filters`, `.sidebar`, `aside` |

## 🧪 БЫСТРЫЙ ТЕСТ

### **Вставь этот код в консоль:**

```javascript
// Тест всех селекторов
const selectors = {
    'Карточка товара': '.product-card:first-of-type',
    'Кнопка Buy': 'button[class*="buy"]'
};

Object.entries(selectors).forEach(([name, selector]) => {
    const el = document.querySelector(selector);
    console.log(
        el ? `✅ ${name}: НАЙДЕН` : `❌ ${name}: НЕ НАЙДЕН`,
        selector
    );
});
```

## 💡 ПОДСКАЗКИ

✅ **Используй простые селекторы:**
- `.product-card` лучше чем `.container > div > div.card`

✅ **`:first-of-type` для первого элемента:**
- `.product-card:first-of-type` → первая карточка

✅ **`[class*="buy"]` для частичного совпадения:**
- Найдет `buy-btn`, `btn-buy`, `buy-now`

✅ **Проверяй в консоли:**
```javascript
document.querySelector('ТВОЙ_СЕЛЕКТОР')
```
