# Портфолио — письма и поп-апы

Сборник шаблонов для портфолио.

## Как работает

1. Ты даёшь шаблон (письмо или поп-ап)
2. Я создаю файл, называю его и сохраняю здесь
3. Собираем всё в одной папке

## Структура

- **Письма** — `.html` файлы
- **Поп-апы** — `.js` или `.html` файлы

## Инструкция

**[POPUP-INSTRUCTION.md](POPUP-INSTRUCTION.md)** — подробная инструкция по поп-апам:
- SVG-крестик, верстка, стили
- События Carrot Quest (открытие, закрытие, отправка)
- LocalStorage, URL-фильтр, SPA, exit intent
- Валидация форм, чек-лист перед деплоем
